# Planner Injector 插件

为 MaiBot 注入用户好感度信息的插件。

## 功能

- 在规划器中注入用户好感度和态度信息
- 智能缓存管理，减少 API 调用
- 提供调试命令用于测试

## 安装

1. 将 `Planner_injector` 文件夹复制到 `plugins` 目录
2. 重启 MaiBot

## 使用

- 给真寻安装[ThelevenFD/zhenxun\_plugin\_MaimAdapter: 用于让maim获取真寻的好感度](https://github.com/ThelevenFD/zhenxun_plugin_MaimAdapter)
- 在配置里填入你的真寻连接地址
